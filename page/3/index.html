如何控制模型

左侧有各种操作工具栏，先点击对应工具，然后模型上面会出现坐标轴等，鼠标可以按住操作。

依次是框选工具，游标工具，移动工具，旋转工具，缩放工具，变换工具

Shift + A 新建物体
G 移动模型， 移动时点击x，会在x轴移动，y和z轴同理。在G移动模式下，按住ctrl移动，可以吸附网格进行移动
S 缩放模型， 缩放是点击x，会在x轴缩放，y和z轴同理；先S，然后按住Shift + x/y/z，是禁止在当前方向上缩放，在另外两个方向上缩放；尽量不要再物体模式下缩放，而是在编辑模式下缩放。如果在物体模式下缩放了，需要按ctrl + a 应用缩放，不然其他倒角等操作会出现异常。

R 旋转模型， 按住x,y,z操作同上
H 隐藏物体

**认识界面

什么是游标和原点

游标

模型中间圆圈叫做游标，两个作用
1。游标在哪里，你新建的物体就会出现在哪里，物体也可以移动到游标
2。游标可以作为物体旋转的轴心点使用

操作游标
点击左侧游标工具，然后鼠标左键点击到哪里游标就到哪里
快捷键：
Shift + 鼠标右键 点击到哪里游标就到哪里
Shift + S 选中物体之后，出现一个操作轮盘，选择到游标，物体就会移动到游标所在的位置
Shift + c 游标回到坐标系原点


默认物体旋转是按照每个物体的原点旋转，可以在变换轴心点中选择游标，之后会按照游标进行旋转

原点
鼠标点击物体选中之后，物体会出现一个黄色的点，就是物体的原点
物体缩放是从原点开始的，可以通过勾选下面的原点，移动物体的原点（移动快捷键G）

可以将原点移动到坐标系中心，之后取消物体移动（Alt + G），可以发现物体的原点回到了世界中心，物体本身和原点还保持着之前的相对关系，但是并没有在世界中心

也就是说程序只认识物体的原点，所有的操作都是按照原点操作

将原点移动到物体默认位置： 右键物体 -> 设置原点->原点到几何中心

在G移动模式下，按住ctrl移动，可以吸附网格进行移动

1, 边界框中心： 选中多个物体是，物体被最小的框包围时，框的中心点
2, 3D游标：坐标系上的圆圈
3, 各自的原点： 每个物体有自己的原点, 认为设置的橙色的点，
4, 质心点： 物体的重心，如果有多个物体是多个物体的重心（这里貌似blender翻译错误，英文是median point, 像是中心点，但不是质心点，实测不是中心，将原点移出物体之外，选择按照质心点旋转，仍然是按照原点旋转）
5, 活动元素： 选中多个元素时，最后选中的那个物体

坐标

1, 全局坐标，所有物体共有的坐标，界面上显示的x, y和z轴
2, 局部坐标，每个物体自己的坐标，跟随物体

快捷键
g x/y/z 局部坐标x/y/z
g x/y/z x/yz 全局坐标x/y/z

四种预览方式

1, 渲染预览，显示最终渲染结果预览，但是相对占资源，慢
2, 材质预览，可以预览材质的颜色和简单纹理，但是不计算光线追踪细节，比渲染预览快
3, 实体模式预览，不现实颜色和材质，建模时候使用，快，不占太多资源
4, 线框模式预览，可以显示内部和背面的走线，建模时候常用

FFmpeg 视频压缩

基本命令

参数说明

参数 说明
-c:v libx264 视频编码用 H.264
-crf 32 压缩质量，越小质量越高。23=视觉无损，32=高压缩
-preset fast 编码速度，fast 比默认 medium 快，文件略大
-c:a aac 音频用 AAC 编码
-b:a 64k 音频比特率 64kbps
-vf "scale=720:-2" 分辨率限制 720p，-2 让宽度自动计算
-fs 48M 文件大小上限，到此自动截断

CRF 参考值

  • 0 = 无损
  • 18 = 近乎无损
  • 23 = 默认，视觉无损
  • 28 = 中等压缩
  • 32 = 高压缩（够用）
  • 51 = 最差

实际案例

小Lin说视频压缩：124MB → 40MB（压缩 68%），CRF 32 + 720p

3.3 修改器

为什么使用修改器

  1. 建模快，效率高，能快速直观地实现一个效果
  2. 非破坏性。可以在不改变原来模型的基础上，很精地控制参数
  3. 灵活，可重复使用，可叠加多个效果。需要说明的是修改器应用是从上到下应用的，顺序会影响结果。

修改器

1 表面细分修改器

将网格的面分割成更小的面，使其看起来更平滑，可以修改修改器的参数使其表面更加平滑


给一个立方体增加一个表面细分修改器，其变成一个球形了，tab进入修改模式可以看到实际还是一个立方体

2 实体化修改器

获取任意网格的表面，然后为之添加深度，使之变厚


新建一个平面，为其添加实体化修改器，修改厚度参数。

3 倒角修改器

宽度是坡口形成的两条新边的距离，段数是增加细分。

4 布尔修改器

可以对两个物体进行交集、差集、并集运算，偶尔会出现奇怪的问题，慎用

给立方体增加一个布尔修改器，然后物体使用吸管选择圆柱体，之后隐藏圆柱体，就能看到差集效果，另外还有并集和交集两种可选

曲线修改器

#贝塞尔曲线

通过两个轴来控制曲线走向


曲线闭合方法：
1, 选择两个点，f, 可以将两个点连接起来，曲线就闭合起来了
2, 选择曲线上的一个点，Alt + c

曲线断开方法：
选择两个点，x，删除分段数，会删除两个点中间的线。不能删除点，删除线之后曲线会自动变小，但是还是闭合的

实现一个管子

通过设置曲线的深度，使曲线形成一个管状，然后可以对其进行挤出旋转等操作，形成一个管子


Alt + s 缩放管子粗细

倒角- 物体

设置的物体相当于曲线的横截面



然后通过缩放，挤出等操作，可以做出动漫人物的头发等

#圆环

圆环就是一个闭合的贝塞尔曲线

#NURBS曲线

Non-Uniform Rational B-Spline 非均匀有理B样条
通过一些点来拟合，控制曲线的走向


将点e挤出，曲线会跟着线拟合

#NURBS圆环

NURBS生成的一个圆，也是NURBS曲线的闭合

#路径曲线

线的开头结尾和点重合


通过移动线上的点，拟合出一条曲线

#曲线修改器

默认简易型便修改器，跟着原点旋转，曲线修改器可以让物体跟着曲线进行形变

注意的点：
路径曲线是有方向的，新建时默认朝向x正轴，物体会沿着路径的的方向进行形变，有方向。

旋转时，正数是顺时针旋转，负数时逆时针旋转

#父子级关系详解

将物体b绑定为物体a的子级：
1, 先选中b
2, 然后选中a
3, Ctrl + p，选择物体

之后移动a(也就是父物体)，子级会跟着移动

一个子级只可以有一个父级，一个父级可以有很多子级
