FFmpeg 视频压缩

基本命令

参数说明

参数 说明
-c:v libx264 视频编码用 H.264
-crf 32 压缩质量，越小质量越高。23=视觉无损，32=高压缩
-preset fast 编码速度，fast 比默认 medium 快，文件略大
-c:a aac 音频用 AAC 编码
-b:a 64k 音频比特率 64kbps
-vf "scale=720:-2" 分辨率限制 720p，-2 让宽度自动计算
-fs 48M 文件大小上限，到此自动截断

CRF 参考值

  • 0 = 无损
  • 18 = 近乎无损
  • 23 = 默认，视觉无损
  • 28 = 中等压缩
  • 32 = 高压缩（够用）
  • 51 = 最差

实际案例

小Lin说视频压缩：124MB → 40MB（压缩 68%），CRF 32 + 720p

3.3 修改器

为什么使用修改器

  1. 建模快，效率高，能快速直观地实现一个效果
  2. 非破坏性。可以在不改变原来模型的基础上，很精地控制参数
  3. 灵活，可重复使用，可叠加多个效果。需要说明的是修改器应用是从上到下应用的，顺序会影响结果。

修改器

1 表面细分修改器

将网格的面分割成更小的面，使其看起来更平滑，可以修改修改器的参数使其表面更加平滑


给一个立方体增加一个表面细分修改器，其变成一个球形了，tab进入修改模式可以看到实际还是一个立方体

2 实体化修改器

获取任意网格的表面，然后为之添加深度，使之变厚


新建一个平面，为其添加实体化修改器，修改厚度参数。

3 倒角修改器

宽度是坡口形成的两条新边的距离，段数是增加细分。

4 布尔修改器

可以对两个物体进行交集、差集、并集运算，偶尔会出现奇怪的问题，慎用

给立方体增加一个布尔修改器，然后物体使用吸管选择圆柱体，之后隐藏圆柱体，就能看到差集效果，另外还有并集和交集两种可选

蒙皮修改器

曲线修改器

#贝塞尔曲线

通过两个轴来控制曲线走向


曲线闭合方法：
1, 选择两个点，f, 可以将两个点连接起来，曲线就闭合起来了
2, 选择曲线上的一个点，Alt + c

曲线断开方法：
选择两个点，x，删除分段数，会删除两个点中间的线。不能删除点，删除线之后曲线会自动变小，但是还是闭合的

实现一个管子

通过设置曲线的深度，使曲线形成一个管状，然后可以对其进行挤出旋转等操作，形成一个管子


Alt + s 缩放管子粗细

倒角- 物体

设置的物体相当于曲线的横截面



然后通过缩放，挤出等操作，可以做出动漫人物的头发等

#圆环

圆环就是一个闭合的贝塞尔曲线

#NURBS曲线

Non-Uniform Rational B-Spline 非均匀有理B样条
通过一些点来拟合，控制曲线的走向


将点e挤出，曲线会跟着线拟合

#NURBS圆环

NURBS生成的一个圆，也是NURBS曲线的闭合

#路径曲线

线的开头结尾和点重合


通过移动线上的点，拟合出一条曲线

#曲线修改器

默认简易型便修改器，跟着原点旋转，曲线修改器可以让物体跟着曲线进行形变

注意的点：
路径曲线是有方向的，新建时默认朝向x正轴，物体会沿着路径的的方向进行形变，有方向。

旋转时，正数是顺时针旋转，负数时逆时针旋转

#父子级关系详解

将物体b绑定为物体a的子级：
1, 先选中b
2, 然后选中a
3, Ctrl + p，选择物体

之后移动a(也就是父物体)，子级会跟着移动

一个子级只可以有一个父级，一个父级可以有很多子级

可以在任意一个点上进行挤出、移动、放大缩小等操作

缩放 Ctrl + a
