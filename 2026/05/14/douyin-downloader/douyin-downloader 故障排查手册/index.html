douyin-downloader 故障排查手册

这篇笔记解决什么问题

这篇笔记用于排查 douyin-downloader 运行和开发中最常见的问题。

建议先读 douyin-downloader 项目开发文档 建立主流程视角，再用这篇按症状定位问题。

排查顺序固定为：

  1. 先确认配置是否生效
  2. 再确认 Cookie 是否有效
  3. 再确认 URL 是否被正确解析
  4. 再确认 API 是否返回正常数据
  5. 再确认下载、落盘、数据库、manifest 是否正常
