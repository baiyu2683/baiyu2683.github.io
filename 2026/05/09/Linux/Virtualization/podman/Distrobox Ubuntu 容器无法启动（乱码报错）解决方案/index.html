🎯 问题原因（核心）

最常见原因如下：

  1. Podman 使用了 runc 而不是 crun

    • 在 Arch + cgroup v2 下应使用 crun
    • 使用 runc 可能导致容器初始化失败（已知问题）

  2. rootless Podman 未正确配置

    • 缺少 /etc/subuid 和 /etc/subgid 映射

  3. 容器初始化过程中状态损坏

    • 需要删除并重新创建

  4. 环境变量污染（低概率）

    • 如 LC_ALL、TMPDIR 等异常
