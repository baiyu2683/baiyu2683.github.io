安装 gvim（会替换 vim，但命令仍然是 vim，不会影响使用）

在 Arch Linux + Ghostty 中，使用普通 vim（通过 pacman -S vim 安装）时，默认没有系统剪贴板支持（vim –version | grep clipboard 会显示 -clipboard）。

下面是 2026 年最实用的几种解决方案，按推荐顺序排列：

1. 最推荐方案：安装 gvim（简单有效）

应该看到 +clipboard（可能还有 +xterm_clipboard）。

  • unnamedplus：让普通 y（yank）直接复制到系统剪贴板（”+ 寄存器，对应 Ctrl+V 粘贴）。
  • 如果你只想用 “+y 手动复制，可以不加这行。

保存后重启 vim 测试。

2. 如何在 Vim 中复制到系统剪贴板

方式 A（推荐，设置后最方便） 选中文字后直接按 y 或 yy（复制当前行），内容就会自动进入系统剪贴板，可以在浏览器、其他终端等地方 Ctrl + V 粘贴。

方式 B（不改配置，手动指定）

  • 进入可视模式（按 v 或 V）
  • 选中想要的内容
  • 按 “ + y （双引号 + 加号 + y）

其他常用命令：

  • 复制当前行到系统剪贴板：”+yy
  • 复制到行尾：”+y$
  • 从系统剪贴板粘贴：”+p 或 “+P
