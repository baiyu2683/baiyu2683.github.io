编辑

编辑模式的切换

为了改变物体的点线面，需要进入编辑模式

快捷键：

功能 键位 补充说明
进入编辑模式 tab 按一次进入编辑模式，再按一次退出

物体模式有这么多，如何快速切换？

打开编辑-> 偏好设置 -> 键位映射，勾选拖动激活饼菜单
之后按住tab键，移动鼠标会出现模式切换的饼菜单，然后鼠标滑动就能快速切换

编辑模式下点线面的切换

模式切换旁边是依次是点模式、线模式、面模式
快捷键

功能 快捷键 补充
点模式 1
线模式 2
面模式 3
同时选多种模式 shift + 1/2/3 也可以鼠标点击多个

点线面的选择方法快捷键和操作截图

功能 快捷键 补充
框选 b
刷选 c 按住鼠标中键可以取消选择，鼠标右键点击空白处退出刷选，或者点击esc键也能退出
套索选择
选择两个点/线/面中间所有的点/线/面 ctrl + 鼠标点击 按照ctrl键，然后鼠标点击选择两个点，这两个点中间最短的路径上所有的点都会被选中。线和面同理。
选择相临的点/线/面 L 会把跟选中的点能连起来的点/线/面都选中，计算是按照四边形计算的，比如四边形和三角形不算相邻。另外鼠标需要在物体上，如果鼠标在空白处也是无法选中的
循环选择 alt + 鼠标点击 选择球体上的一个点，然后鼠标点击点的左或者右，就会按照纬线选中一圈；点击上下，就会按照经线方向选中一圈。 线模式和面模式类似
在垂直方向上循环选择 ctrl + alt + 鼠标点击 选中垂直于选中线的一圈元素
反选 ctrl + i 选中所有没有被选中的元素
随机选择 随机选择一些点线面
扩大/减小选取 ctrl + 数字键盘+/- 需要有小键盘
透视模式 alt + z 默认情况选择只能选到看见的一面，透视模式可以选到背面的


按住ctrl 选中了两个红圈的点，中间所有的点都被自动选中了


选中四边面的一个点，按l无法选中三角面上的点，需要鼠标放到三角面上再按l才能选中


在线模式下，然后使用ctrl + alt + 鼠标左键点击可以选中垂直于线的一圈线


随机选择任意点线面


透视模式选择球体的线

点线面的控制

操作 快捷键 补充
移动 g
旋转 r
缩放 s
删除 x 会弹出菜单

法向： 是一个向量的概念，有方向，为了区分面的正反而设计的概念，物体的正面就是法向。法线是沿着法向方向的，只有正面有


打开这个面朝向选项可以看到面的正反面



如上面两张图，红色是反面，默认颜色是正面

法线：垂直于元素的线，只在物体正面有，是为了告诉渲染器光的反射角度，点线面都有

通过这个开关可以显示法线

法线只显示在正面，也就是法向指向的方向
