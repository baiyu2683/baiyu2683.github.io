isAssignableFrom() 是 Java 反射中非常重要且常用的一个方法，其作用是：

判断一个类是否可以被另一个类赋值（即类型兼容性检查）

官方定义

Java

含义： 判断 cls 这个类（或接口）是否能赋值给当前 Class 对象所代表的类型。

简单说就是：“cls 是否是当前类型的子类、实现类、或者就是当前类型本身？”
