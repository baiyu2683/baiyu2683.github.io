3.9.1 指数记数法

Created2026-05-09|Updated2026-05-09
|Post Views:

我记得指数记数法是有一个国际标准的，小数点多少位，指数多少位，怎么记录这些数字？

单精度和多精度（通常指双精度、四倍精度等）数字表示法的国际标准是 IEEE 754（全称 IEEE Standard for Floating-Point Arithmetic）。

该标准由电气电子工程师学会 (IEEE) 制定，并被国际标准化组织 (ISO) 和国际电工委员会 (IEC) 采纳为国际标准，编号为 ISO/IEC/IEEE 60559。

📋 标准核心内容

IEEE 754 规定了计算机中浮点数的存储格式、算术运算、舍入规则以及异常处理（如溢出、被零除、NaN 等）。

1. 常见精度格式

在 IEEE 754 标准中，最常用的两种格式是单精度和双精度：

格式名称 别名 总位数 符号位 (Sign) 指数位 (Exponent) 尾数位 (Mantissa) 十进制精度
单精度 binary32 32 bit 1 bit 8 bit 23 bit ~7 位有效数字
双精度 binary64 64 bit 1 bit 11 bit 52 bit ~16 位有效数字
四倍精度 binary128 128 bit 1 bit 15 bit 112 bit ~34 位有效数字

2. 数值的表示公式

一个浮点数的值通过以下公式计算：

$$Value = (-1)^S \times (1.f) \times 2^{E - Bias}$$

  • $S$ (Sign)：符号位。0 代表正数，1 代表负数。

  • $f$ (Fraction/Mantissa)：尾数的小数部分。隐含的高位为 1（规格化数）。

  • $E$ (Exponent)：阶码（指数）。

  • $Bias$ (偏移量)：为了表示负指数，标准规定了一个偏移值。单精度为 127，双精度为 1023。

🔍 为什么这个标准很重要？

  1. 跨平台一致性：无论你在 Intel CPU、ARM 架构还是英伟达 GPU 上运行，同样的浮点运算代码应该得到相同的结果。

  2. 特殊的数值处理：标准定义了如何表示特殊状态：

    • NaN (Not a Number)：用于非法运算（如 $\sqrt{-1}$）。

    • Infinity ($\infty$)：用于表示超出范围的极值。

    • $\pm 0$：区分正零和负零。

  3. 舍入模式：规定了四种（或更多）舍入规则，确保精度损失是可控且可预测的。
